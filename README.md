# Cómputo en la Nube – Tareas y Prácticas

Este repositorio contiene las **tareas, ejercicios y prácticas** desarrolladas durante la materia **Cómputo en la Nube**, como parte del programa de maestría.

## Descripción
El objetivo de este repositorio es centralizar las implementaciones realizadas a lo largo del curso, abordando conceptos fundamentales del cómputo en la nube, diseño de arquitecturas básicas, programación paralela y concurrente, así como el uso de servicios comunes de plataformas de nube.

Cada tarea busca aplicar de manera práctica los conceptos teóricos vistos en clase, con énfasis en soluciones escalables y eficientes.

## Objetivos del curso
- Comprender los fundamentos del cómputo en la nube.
- Diseñar arquitecturas básicas de soluciones en la nube.
- Aplicar conceptos de concurrencia y paralelismo.
- Implementar soluciones básicas utilizando servicios de nube.
- Analizar ventajas y desventajas entre infraestructuras locales y en la nube.

## Estructura del repositorio
El repositorio está organizado por tareas:

```
├── tarea-1-programacion-paralela/
├── tarea-2-...
├── tarea-3-...
└── README.md
```

Cada carpeta incluye el código fuente, archivos de configuración y, cuando aplica, documentación adicional.

## Tecnologías y herramientas
- Lenguajes de programación: C / C++ / Python (según la tarea)
- Programación paralela: OpenMP
- Entornos de desarrollo y herramientas de nube (según el alcance de cada actividad)

## Contexto académico
Repositorio desarrollado con fines **estrictamente académicos** como parte de la materia **Cómputo en la Nube** del Tecnológico de Monterrey.
